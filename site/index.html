✂ Schnipsel

Collect pieces of the web. Arrange them. Search them. Share them.

Schnipsel is a Firefox extension that clips fragments of any web page — keeping their original layout and styling — and organises them into named bags you can collage, search, and share peer-to-peer.

Install

  1. Download the latest .xpi from Releases.
  2. In Firefox, open about:addons → gear icon ⚙ → Install Add-on From File…
  3. Click the ✂ button in the toolbar to open the sidebar.

Once installed, Schnipsel updates itself automatically when a new version is released.
